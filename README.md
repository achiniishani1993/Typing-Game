# Typing Speed Game

A simple browser-based typing game built with **HTML, CSS, and JavaScript**.  
Type the displayed word correctly before time runs out to score points and extend your time!

---

## Features

* Random word generation
* Score tracking system
* Countdown timer
* Difficulty levels (Easy / Medium / Hard)
* Game over screen with restart button
* Simple settings panel
